Objetivo: Análisis comparativo de modelos predictivos para acciones del mercado financiero

Descripción del Proyecto

Este libro presenta un análisis exhaustivo de diferentes modelos predictivos aplicados a una selección de acciones del mercado financiero. Cada capítulo se enfoca en un ticker específico, proporcionando:

Modelos Evaluados

  • ARIMA (AutoRegressive Integrated Moving Average): Modelo de series temporales que captura tendencias y patrones estacionales
  • Monte Carlo: Simulación estocástica para modelar múltiples escenarios futuros
  • Regresión: Análisis de relaciones lineales/no-lineales entre variables
  • SMA (Simple Moving Average): Media móvil simple para identificación de tendencias

Métricas de Evaluación

Para cada modelo se calculan las siguientes métricas de rendimiento:

  • MAE (Mean Absolute Error): Error absoluto medio
  • RMSE (Root Mean Square Error): Raíz del error cuadrático medio
  • MAPE (Mean Absolute Percentage Error): Error absoluto porcentual medio
  • R² (Coefficient of Determination): Coeficiente de determinación

Análisis de Riesgo

Cada capítulo incluye un análisis completo de métricas de riesgo calculadas para diferentes horizontes temporales:

  • Volatilidad: Medida de dispersión de retornos
  • VaR (Value at Risk): Pérdida máxima esperada con nivel de confianza determinado
  • Expected Shortfall: Pérdida promedio en escenarios extremos
  • Drawdown: Caída máxima desde pico hasta valle

Estructura del Documento

Cada capítulo de acción contiene:

  1. Gráfico Principal: Comparación visual de todos los modelos
  2. Tablas de Resultados: Valores históricos vs predicciones
  3. Análisis por Modelo: Visualizaciones específicas de cada enfoque
  4. Métricas de Evaluación: Comparación cuantitativa de rendimiento
  5. Gestión de Riesgos: Análisis de métricas de riesgo por frecuencia

Tecnologías Utilizadas

library(forecast)     # Modelos ARIMA
